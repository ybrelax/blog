## 文件命名规范

### 项目命名

全部采用小写方式， 以下划线分隔。

例：my-project-name

### 目录命名
参照项目命名规则；

有复数结构时，要采用复数命名法。

例：scripts, styles, images, data-models

### JS/TS文件命名
参照项目命名规则。

例：account-model.js account-model.ts

### CSS, SCSS文件命名
参照项目命名规则。

例：retina-sprites.scss

## HTML

### HTML文件命名
参照项目命名规则。

例：error-report.html
