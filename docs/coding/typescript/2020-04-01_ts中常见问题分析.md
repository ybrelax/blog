### 推荐module 改为 namespace

新版本中推荐我们用 namespace 来代替 module

module: 
```ts
module Person {
    export xxx
}
```
namespace:
```ts
module Person {
    export xxx
}
···

### null 和 undefined是其他类型的（包括void)的子类型，可以赋值给其他类型的

* 默认情况下，编译器会提示错误，这是因为 tsconfig.json 里面有一个配置项是默认开启的。
```json
// tsconfig
{
  "strict": true // 可以关闭这个严格模式
}
```
